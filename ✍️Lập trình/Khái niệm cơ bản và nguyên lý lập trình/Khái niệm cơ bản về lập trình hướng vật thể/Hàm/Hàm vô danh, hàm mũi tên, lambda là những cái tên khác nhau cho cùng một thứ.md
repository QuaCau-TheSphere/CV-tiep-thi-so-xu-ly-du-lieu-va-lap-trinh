Ví dụ hàm này:
```js
function (x){
    return y
}
```

Có thể được viết dưới dạng mũi tên như sau:
```js
x => y
(x) => y
(x) => {return y}
```

Trong Python thì nó gọi là lambda:
```python
lambda x: y
```

Những hàm như thế này sẽ được dùng cho [[Callback là những hàm được dùng như đối số của hàm khác|callback]]
