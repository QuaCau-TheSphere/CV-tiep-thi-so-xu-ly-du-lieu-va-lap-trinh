| Lệnh                        | Ý nghĩa                                            |
| --------------------------- | -------------------------------------------------- |
| `git log`                   | Xem các danh sách các commit                       |
| `git log --oneline`         | Chỉ xem tên                                        |
| `git log --oneline --graph` | Xem sơ đồ các commit                               |
| `git log <commit>`          | Xem danh sách các commit từ `<commit>` đó về trước |
|                             |                                                    |
