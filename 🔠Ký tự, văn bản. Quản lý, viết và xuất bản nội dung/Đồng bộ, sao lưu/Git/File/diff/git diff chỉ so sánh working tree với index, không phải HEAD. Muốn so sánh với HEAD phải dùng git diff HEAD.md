## So sánh cùng một tệp giữa hai commit
```bash
git diff <commit 1> <commit 2> -- <file>
```
## So sánh hai tệp khác nhau giữa hai commit
```bash
git diff <commit 1>:<file 1> <commit 2>:<file 2>
```
